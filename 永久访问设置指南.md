## 📋 最简单的5步设置（5分钟完成）

### 第1步：创建GitHub账号（如果您还没有）
1. 访问：https://github.com
2. 点击右上角绿色 "Sign up" 按钮
3. 填写：
   - Email: 您的邮箱
   - Password: 设置密码
   - Username: 建议用英文，如 `haohaolearn` 或 `haohaomath2024`
4. 验证邮箱（点击邮件中的链接）
5. 完成注册

### 第2步：创建仓库
1. 登录GitHub后，点击右上角 "+" 号
2. 选择 "New repository"
3. 填写仓库信息：
   ```
   Repository name: hao-hao-math-learning
   Description: 浩浩数学学习乐园
   ☑️ Public (必须选择公开)
   ⬜ 不要勾选 Add a README file
   ```
4. 点击底部绿色按钮 "Create repository"

### 第3步：推送代码到GitHub
创建仓库后，GitHub会显示一个页面，复制并运行下面的命令：

**方式A：使用自动化脚本（推荐）**
```
双击项目文件夹中的 "部署到GitHub.bat"
```

**方式B：手动命令行**
```
git remote add origin https://github.com/您的用户名/hao-hao-math-learning.git
git branch -M main
git push -u origin main
```

### 第4步：启用GitHub Pages
1. 在您的仓库页面，点击顶部 "Settings" 标签
2. 在左侧菜单中找到并点击 "Pages"
3. 在 "Build and deployment" 部分设置：
   ```
   Source: 选择 "Deploy from a branch"
   Branch: 选择 "main" 分支
   Folder: 选择 "/ (root)"
   ```
4. 点击蓝色 "Save" 按钮

### 第5步：获取永久地址
1. 等待2-3分钟（GitHub正在部署）
2. 在Pages设置页面顶部会显示：
   ```
   🎉 Your site is live at:
   https://您的用户名.github.io/hao-hao-math-learning
   ```
3. 这个地址就是您的永久访问地址！

## 🆘 遇到问题？

### 问题1：推送失败
**解决**：检查GitHub用户名是否正确，仓库是否已创建

### 问题2：Pages无法访问
**解决**：
- 确认仓库是Public（公开）
- 等待5-10分钟让GitHub完成部署
- 检查Pages设置是否保存成功

### 问题3：语音功能不能用
**解决**：GitHub Pages默认使用HTTPS，应该可以正常使用
