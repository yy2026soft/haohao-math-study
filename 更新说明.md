## 🚀 如何使用

### 正常答题流程
1. 看题目 → 输入答案 → 提交
2. **答对**：
   - 听到"浩浩小朋友你真棒！"
   - 1.5秒后自动显示下一题
   - 继续答题

3. **答错**：
   - 听到"浩浩，再试试看！"
   - 答案框自动清空
   - 重新尝试当前题目

### 语音答题流程
1. 点击🎤按钮 → 允许麦克风权限
2. 大声说出答案（如："五"或"5"）
3. 系统识别并填入 → 自动提交
4. 按上述流程继续

## 💡 使用建议

### 给家长的建议
1. **推荐使用Chrome浏览器**：语音识别效果最好
2. **安静环境**：让浩浩在安静的地方答题
3. **鼓励为主**：答错时鼓励再试试，不要批评

1. **大声说话**：使用语音时要说清楚
2. **认真思考**：不要着急，想好了再回答
3. **多练习**：每天坚持练习，进步更快
4. **听语音提示**：仔细听语音，知道自己做得怎么样
