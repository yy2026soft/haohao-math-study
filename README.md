# 浩浩数学学习乐园

## 🌟 项目介绍
这是一个专为儿童设计的数学学习应用，包含加法、减法、乘法、除法练习，以及学习记录管理和导出功能。

## 🚀 快速开始

### 方法1：直接本地运行
1. 下载项目文件到本地
2. 双击 `index.html` 文件，即可在浏览器中运行

### 方法2：使用本地服务器
```bash
# 使用Python启动服务器
python -m http.server 8080

# 或使用Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 📱 功能特性

### 🎮 学习模块
- **加法探险**：循序渐进的加法练习
- **减法迷宫**：有趣的减法训练
- **乘法城堡**：乘法运算和九九乘法表
- **除法海洋**：除法运算练习
- **九九乘法表**：可视化乘法口诀学习，支持语音朗读

### 🎤 语音功能
- 智能语音识别，支持中文数字输入
- 自动语音反馈，增强学习体验
- 支持各种数字表达方式（"五"、"5"、"五十五"等）

### 📊 学习记录
- 详细记录每次学习情况
- 按日期统计学习成果
- 支持数据筛选（今天、本周、本月）
- **Excel报表导出**：一键导出学习记录为Excel文件

### 🏆 激励系统
- 印章积分奖励机制
- 实时学习反馈
- 成就统计展示

## 📋 技术栈
- **前端**：纯HTML5 + CSS3 + JavaScript
- **语音识别**：Web Speech API
- **数据存储**：LocalStorage（本地存储）
- **导出功能**：XLSX.js（Excel导出）

## 🔧 部署指南

### GitHub Pages部署
1. 将项目上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择main分支作为源
4. 即可通过 `https://username.github.io/repo` 访问

### 其他免费托管服务
- **Netlify**：拖拽文件夹即可部署
- **Vercel**：连接GitHub仓库自动部署
- **Firebase Hosting**：Google提供的免费静态托管

### 本地持久访问
如果需要在本地网络内长期访问：
```bash
# 使用Python启动服务器（在项目目录下）
python -m http.server 8080

# 其他设备通过以下地址访问
# http://您的IP地址:8080
```

## 👨‍👩‍👧 使用说明
1. **选择学习模式**：从主页选择要练习的运算类型
2. **设置难度**：根据孩子的水平选择合适的难度
3. **开始学习**：可以通过键盘输入或语音回答
4. **查看记录**：在"学习记录"页面查看学习情况
5. **导出报表**：点击"导出报表"按钮生成Excel文件

## 📞 技术支持
如遇到问题，请检查：
1. 浏览器是否支持Web Speech API
2. 是否在HTTPS环境下使用语音功能
3. 是否允许浏览器使用麦克风权限
